# Reference Npc

A [fact](/docs/facts) that references an NPC in the Citizens plugin.

## How could this be used?

This fact could be used to reference an NPC in the Citizens plugin. This could be used to create a quest that requires the player to talk to an NPC.

## Fields

<SpeakersField />

<EntryField name="Npc Id" required>
    The id of the NPC in the Citizens plugin.
</EntryField>
