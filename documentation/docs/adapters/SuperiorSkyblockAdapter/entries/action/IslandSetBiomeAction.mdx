# Island Set Biome Action

The `Island Set Biome` action is used to set a player's island's biome.

## How could this be used?

This action could be used to set the biome of a player's island to a specific biome, when they run a command or on some other event.

## Fields

<ActionsField />

<EntryField name="Biome" required>
    The biome to set the island to
</EntryField>
