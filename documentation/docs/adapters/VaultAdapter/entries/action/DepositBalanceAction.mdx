# Deposit Balance Action

The `Deposit Balance Action` is used to deposit money into a user's balance.

## How could this be used?

This action could be used to deposit money into a user's balance when they complete a task.

## Fields

<ActionsField />

<EntryField name="Amount" required>
    The amount of money to deposit.
</EntryField>
