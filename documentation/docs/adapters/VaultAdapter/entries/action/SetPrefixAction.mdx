# Set Prefix Action

The `Set Prefix Action` action sets the prefix of a player's message

## How could this be used?

This action could be used to set the prefix of a player's message if they have a certain amount of money.

## Fields

<ActionsField />

<EntryField name="Prefix" required>
    The prefix to set.
</EntryField>
