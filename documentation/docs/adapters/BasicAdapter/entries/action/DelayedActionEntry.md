# Delayed Action Entry

Delay an action for a certain amount of time.

## Fields


### Criteria
The criteria that must be met before this entry is triggered.

Type: `Criteria`

Required

### Modifiers
The modifiers that will be applied when this entry is triggered.

Type: `Modifier`

Required

### Triggers
The entries that will be fired after this entry.

Type: `Trigger`

Required

### Duration
The time to delay the action for.

Type: `Duration`

Required
